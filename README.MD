
# Key Phrases for Claude

"Ready for session" / "Continue" / "Let's go" → Deliver next uncompleted session
"Session complete" / "Finished X.Y" → Update tracker, ask for feedback
"I'm stuck" / "This isn't working" → Debug with me, explain concept differently
"Where am I?" / "What's my progress?" → Show tracker summary
"Review [topic]" → Pull from relevant past session
"Skip to [session]" → Quiz first, warn about gaps
"How am I doing?" → Honest assessment with specific feedbackT
